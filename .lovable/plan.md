## 4. Технические детали

```text
DB:
  lot_reservations  + bid_amount int, + request_type text
  forum_registrations  (new)
  RLS: anyone INSERT, admin SELECT/UPDATE

Routes:
  + /forum
  /auth — только sign-in
  /lots/:id — без auth-стены

Components:
  + BidRequestModal
  + Forum page sections
  - LotInterestModal
  - TicketsSection (с Index)

Edge functions:
  переиспользуем notify-telegram, send-transactional-email
```
